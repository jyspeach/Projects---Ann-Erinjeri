# Machine Learning for Insurance Policy Prediction

## Overview
This project develops machine learning models to predict whether an existing health insurance customer will purchase a new car insurance policy.

The dataset contains demographic, vehicle-related, and insurance-related attributes. The response variable indicates whether the customer purchased the new policy.

## Objectives
- Perform Exploratory Data Analysis (EDA)
- Handle class imbalance
- Train and evaluate machine learning models
- Compare model performance using appropriate metrics

## Dataset
The dataset includes features such as:

- Age
- Annual Premium
- Vintage (customer tenure)
- Gender
- Driving License
- Vehicle Age
- Previous Insurance
- Vehicle Damage
- Response (target variable)

The dataset exhibits **class imbalance**, where approximately **87.8% did not purchase insurance and 12.2% did**. 

## Methods
The following steps were performed:

1. Exploratory Data Analysis
2. Feature transformation
3. Model training
4. Performance evaluation

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

Due to the class imbalance, **recall and F1-score are particularly important**.
