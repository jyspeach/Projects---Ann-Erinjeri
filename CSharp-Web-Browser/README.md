# C# Web Browser Implementation

## Overview
This project implements a simple web browser using C# and Windows Forms.

The browser demonstrates core networking functionality such as sending HTTP requests, retrieving HTML content, and managing browsing history.

## Features

Fully implemented:

- Sending HTTP requests
- Displaying HTTP response status codes
- Displaying webpage titles
- Reloading pages
- Bookmark management
- History tracking
- Homepage configuration
- Extracting links from webpages

Partially implemented:

- Back/forward navigation across sessions

## System Architecture

The browser is built using:

- Windows Forms GUI
- .NET Framework 4.8
- SQLite database for persistent storage

Key components include:

- History manager
- Bookmark manager
- HTTP request handler
- Link extraction module

## Data Structures

The application uses:

- Lists for browsing history
- Stacks for navigation
- Dictionaries for user profiles
- SQLite database for persistent data storage

## Technologies

- C#
- .NET Framework
- Windows Forms
- SQLite
