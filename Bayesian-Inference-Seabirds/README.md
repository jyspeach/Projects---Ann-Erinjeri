# Bayesian Inference for Seabird Migration

## Overview
This project applies Bayesian probability concepts to model the migration of seabirds between two islands over two years.

The analysis derives probability distributions and conditional relationships between migration events.

## Problem Description
Seabirds may move between two islands:

- Island E (East)
- Island W (West)

The migration probability is governed by parameter δ.

The probability that a seabird is on Island E in year 2 is:

θE = pE(1 − 2δ) + δ

where:
- pE = probability of being on island E in year 1
- δ = switching probability

## Key Tasks
The project includes:

1. Deriving probabilities for island locations in year 2
2. Applying Bayes’ theorem to compute conditional probabilities
3. Deriving marginal distributions
4. Computing conditional distributions
5. Deriving posterior distributions
